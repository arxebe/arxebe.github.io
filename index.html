    PROJECTS


    Aww that body

    Created a web platform that allows users to track their food intake. The web platform was built using node.js, express.js as the backend server and react.js as the frontend framework. The platform also uses chart.js to create charts and dashboard that display to the users their dietary statistics. MongoDB was used as the database to support the different entities.     Do-Do Bird

    Do-Do Bird is an intelligent personal assistant that organises your tasks for you. Tempo aims to allow users to manage their schedule with just a single line command.     Random Forest Genetic Learning Agent

    This is a Tetris Learning Agent that was created for the AI module in NUS. I used a Genetic Learning algorithm as the training algorithm to train the weights of a heuristic function that score the state in the game of Tetris. However, a single estimator has a huge variance, hence, I adapted the random forest technique to train multiple heuristic functions and combine their score through weighted vote.
